## :floppy_disk: Installation

<h4>Follow the steps bellow to run this locally on your machine: </h4>  

1. Clone the repo
   ```sh
   git clone https://github.com/rafAmorim8/toDoList.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

3. Run the project
```sh
   npm run dev
   ```
